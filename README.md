# Student Dropout Analysis in Colombia

## *Why is student dropout so high in Colombian higher education?*

## Project Overview

This project aims to analyze student dropout rates in Colombian higher education institutions using open government data. The objective is to identify patterns, trends, and potential factors associated with student attrition through data analysis and interactive dashboards.

## Tools

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Socrata Open Data API
- Git & GitHub
